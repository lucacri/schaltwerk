## What is Lucode?

Run multiple AI agents on the same codebase—safely. Lucode is organized around **tasks**: each task owns an isolated [git worktree](https://git-scm.com/docs/git-worktree) with its own branch, and stage runs spawn slot sessions where agents do focused work.

## Quick Start (1 Minute)

<Steps>
  <Step title="Install">
    ```bash
    git clone https://github.com/lucacri/lucode.git
    cd lucode && bun install && just install-fast
    open /Applications/Lucode.app
    ```
  </Step>

  <Step title="Open your project">
    Select your git repository in Lucode
  </Step>

  <Step title="Create a task">
    Press `⌘N` to create a new task:
    - Enter a task description (this becomes the task’s Spec artifact)
    - Select your agent (GitHub Copilot, Claude Code, OpenCode, Codex, Gemini, Kilo Code, Qwen, Factory Droid, Amp, or Terminal Only)
    - Choose the base branch
    - Press `⌘Enter` to start
  </Step>

  <Step title="Watch it work">
    The task appears in the sidebar with its own isolated [git worktree](https://git-scm.com/docs/git-worktree) and branch. Stage runs spawn slot sessions where the agent works; monitor progress in the top terminal.
  </Step>

  <Step title="Review and merge">
    When done, merge to main with `⌘⇧M` or create a PR with `⌘⇧P`.
  </Step>
</Steps>

## Key Features

- **Isolated Git Worktrees** - Each task owns a base worktree and branch; slot sessions run underneath
- **Stage-Driven Flow** - Tasks move through stages (Draft → Ready → Brainstormed → Planned → Implemented → Pushed → Done) at your pace
- **Dual Terminals** - Agent terminal on top, your shell below
- **Spec / Plan / Summary Artifacts** - Each task carries its planning docs alongside the worktree
- **Orchestrator** - A singleton non-task surface in your primary branch for coordination work
- **Web Preview** - View localhost dev servers directly in the app with auto-detection and element inspection
- **MCP API** - Automate task management via REST
